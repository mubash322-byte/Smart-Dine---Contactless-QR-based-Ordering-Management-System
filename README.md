# Smart-Dine---Contactless-QR-based-Ordering-Management-System
The Smart Dine System is an innovative restaurant management solution designed to digitalize the traditional dining experience. The main objective of this project is to simplify the process of ordering food in restaurants by using QR code-based technology. Each dining table in the restaurant is assigned a unique table name and QR code. When customers scan the QR code using their smartphones, they are directed to the restaurant’s online menu interface.  From this interface, customers can conveniently browse the menu, select their preferred dishes, and place orders directly from their table without waiting for a waiter to take the order manually. Once the order is confirmed, it is automatically sent to the kitchen system and waiter’s dashboard, where it can be prepared and served efficiently.
